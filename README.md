# Web Engineering 2022 - Solution Navigator
Dies ist das Repository für die Abgabe im Kurs Web Engineering 2022 von Lukas Busse

## Aufgabenbeschreibung
Die Aufgabe im Kurs Web Engineering war, einen sog. Solution Navigator zu entwickeln, auf der Grundlage von HTML, CSS und JavaScript. Ohne dabei zusätzliche Frameworks zu verwenden.
Der Solution Navigator soll die einzelnen wöchentlichen Abgaben vorstellen.
